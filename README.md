## Tech Stack 
- Flutter (latest version) 
- Dart

## Installation 
1.	Clone the repository: 
git clone https://github.com/shtwaijri/Learning-App-UI.git

2.	Navigate to the project folder: 
cd  Learning-App-UI

3. Install dependencies: 
flutter pub get 

4.	Run the app: 
flutter run
